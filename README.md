# Internetspeed

Open-source CLI internet speed tool.

## Features

- Download speed
- Upload speed
- Public IP detection
- Live monitoring
- CLI interface

## Install

### From GitHub

```bash
pip install git+https://github.com/YOUR_USERNAME/internetspeed.git
```

## Usage

```bash
internetspeed test
internetspeed ip
internetspeed live
```
